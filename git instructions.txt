---------------- FAST START de REPO sur le machine locale ---------------

1. Allez a le dosier ou vous vouler avoir votre repo
2. Right click -> Git Bash here (en PC, sais pas en Mac)
3. dans le bash:	- git init
4. maintenant on peut trouver le dossier [!git] et tout le contenu est suivi par git.
 on peut y mettre tous ce que vous voulez
5. dans le bash:  - "git add ." - ajouter tous les fichiers pour commit
6.                - "git commit -m "commit message"" - git prepare tous les fichiers ajoutes pour garder comme version intermediare
7. maintenant ce repo locale est pret pour etre envoyé sur github
